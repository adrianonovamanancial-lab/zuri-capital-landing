# Zuri Capital - Brainstorm de Design

## Contexto
Landing page para fintech de crédito rápido, focada em conversão imediata via chat de IA. Tema dark premium com verde neon/lima como destaque.

---

## Resposta 1: Minimalismo Corporativo Futurista
**Probabilidade: 0.08**

### Design Movement
Neo-brutalism corporativo com influências de design de interface de ficção científica. Referências: Stripe, Figma, interfaces de startups deep-tech.

### Core Principles
1. **Clareza radical**: Cada elemento serve um propósito específico; nenhum ornamento desnecessário
2. **Hierarquia através de escala**: Tamanhos de fonte e espaçamento definem importância
3. **Contraste extremo**: Preto profundo (#0B0F1A) contra verde lima (#C6D600) cria tensão visual
4. **Tipografia como estrutura**: Fonts geométricas (sans-serif moderna) definem a identidade

### Color Philosophy
- **Fundo**: Preto profundo (#0B0F1A) com sutis gradientes de azul escuro
- **Destaque**: Verde lima (#C6D600) apenas em CTAs e elementos críticos
- **Cinzas**: #2A2A2A para separação, #A0A0A0 para texto secundário
- **Intenção**: Transmitir segurança, tecnologia, e velocidade sem frivolidade

### Layout Paradigm
- Hero assimétrico: título à esquerda, mockup de celular à direita com overflow
- Chat integrado como sidebar flutuante (não modal)
- Seções em blocos verticais com quebras de ritmo (algumas full-width, outras com padding)
- Uso de linhas finas horizontais para separação visual

### Signature Elements
1. **Linhas geométricas animadas**: Linhas que conectam elementos, reforçando fluxo
2. **Mockup de celular com glassmorphism**: Tela do app com efeito de vidro fosco
3. **Ícones customizados**: Formas geométricas simples (círculos, quadrados, linhas)

### Interaction Philosophy
- Hover states sutis: leve aumento de opacidade, não cores
- Cliques abrem transições suaves (fade-in do chat)
- Scroll revela elementos com entrance animations discretas
- Feedback imediato: ripple effect em botões

### Animation
- Entrance: fade-in com slide suave (100ms)
- Hover: opacity change (200ms ease-out)
- Chat open: slide from right (300ms cubic-bezier)
- Scroll animations: stagger effect em listas

### Typography System
- **Display**: Roboto Mono Bold (títulos) - futurista, técnico
- **Body**: Inter Regular (corpo) - legibilidade
- **Peso**: Bold para títulos, Regular para corpo, Medium para labels
- **Tamanho**: 48px (h1), 32px (h2), 16px (body), 14px (small)

## Resposta 2: Dark Elegância Minimalista
**Probabilidade: 0.07**

### Design Movement
Luxo digital com influências de design de joalharia e moda. Referências: Hermès digital, Apple premium, interfaces de wealth management.

### Core Principles
1. **Espaço respeitado**: Muito whitespace (ou em caso, blackspace)
2. **Tipografia elegante**: Fontes com serifa ou sans-serif sofisticada
3. **Sutileza extrema**: Efeitos visuais quase imperceptíveis
4. **Qualidade sobre quantidade**: Poucos elementos, cada um refinado

### Color Philosophy
- **Fundo**: Gradiente sutil de azul-escuro (#0B0F1A) para roxo-escuro (#1a0f2e)
- **Destaque**: Verde lima (#C6D600) em pequenas doses, como joia
- **Cinzas**: Tons quentes (#3A3A3A, #B0B0B0) para humanizar
- **Intenção**: Transmitir sofisticação, confiança, exclusividade

### Layout Paradigm
- Composição assimétrica com golden ratio
- Chat como painel deslizante elegante (overlay com backdrop blur)
- Seções com muito espaço negativo
- Alinhamento vertical com breathing room

### Signature Elements
1. **Gradientes sutis**: Transições de cor imperceptíveis
2. **Tipografia em destaque**: Palavras-chave em verde lima
3. **Linhas finas de ouro/prata**: Separadores minimalistas

### Interaction Philosophy
- Transições lentas e fluidas (400ms+)
- Hover: mudança de cor muito sutil
- Cliques: animação de escala (scale 0.98)
- Feedback: toast discreto no canto inferior

### Animation
- Entrance: fade-in lento (400ms)
- Hover: color shift (300ms ease-in-out)
- Chat open: slide + fade (400ms)
- Scroll: parallax suave em backgrounds

### Typography System
- **Display**: Playfair Display (títulos) - elegante, sofisticado
- **Body**: Lato (corpo) - legível, amigável
- **Peso**: Bold para títulos, Light para subtítulos, Regular para corpo
- **Tamanho**: 56px (h1), 36px (h2), 16px (body), 13px (small)

## Resposta 3: Energético Tech-Forward
**Probabilidade: 0.09**

### Design Movement
Cyberpunk moderno com influências de design de gaming e startups de IA. Referências: Discord, Figma, interfaces de IA generativa.

### Core Principles
1. **Dinamismo visual**: Movimento e energia em cada seção
2. **Cor como informação**: Verde lime para ação, cinzas para contexto
3. **Tipografia bold**: Fontes geométricas com peso alto
4. **Padrões e texturas**: Grids, linhas, padrões geométricos

### Color Philosophy
- **Fundo**: Preto (#0B0F1A) com padrão de grid sutil
- **Destaque primário**: Verde lima (#C6D600) em abundância
- **Destaque secundário**: Tons de ciano/azul (#00D9FF)
- **Intenção**: Transmitir inovação, velocidade, energia, modernidade

### Layout Paradigm
- Grid-based com quebras assimátricas
- Chat como elemento central (não lateral)
- Seções com padrões de fundo animados
- Uso de cards com borders em verde

### Signature Elements
1. **Padrão de grid animado**: Background com linhas que se movem
2. **Badges e pills**: Elementos com borders em verde lima
3. **Ícones com gradientes**: Ícones com verde → ciano

### Interaction Philosophy
- Cliques com efeito de pulse
- Hover: mudança de cor + glow effect
- Chat: animação de entrada com bounce
- Feedback: notificações com cor e som

### Animation
- Entrance: scale + fade (200ms)
- Hover: glow effect (300ms)
- Chat open: bounce + slide (400ms cubic-bezier)
- Scroll: counter animations em números

### Typography System
- **Display**: Space Mono Bold (títulos) - futurista, monoespacial
- **Body**: IBM Plex Sans (corpo) - técnico, legível
- **Peso**: Bold para títulos, Medium para labels, Regular para corpo
- **Tamanho**: 52px (h1), 32px (h2), 16px (body), 14px (small)

## Decisão: Abordagem Selecionada

**Escolhido: Resposta 1 - Minimalismo Corporativo Futurista**

### Justificativa
Para uma fintech focada em conversão e confiança, o minimalismo corporativo futurista oferece:
- ✅ Clareza radical que facilita conversão (usuário sabe exatamente o que fazer)
- ✅ Contraste extremo que destaca CTAs e chat de IA
- ✅ Sensação de tecnologia avançada sem parecer "genérico"
- ✅ Escalável: funciona bem em mobile e desktop
- ✅ Referências (Stripe, Figma) são fintechs/SaaS bem-sucedidas

### Implementação
- **Tipografia**: Roboto Mono para títulos (futurista), Inter para corpo
- **Cores**: #0B0F1A (fundo), #C6D600 (destaque), #2A2A2A e #A0A0A0 (neutros)
- **Layout**: Hero assimétrico, chat como sidebar, linhas geométricas
- **Animações**: Entrance suaves, hover discretos, transições rápidas
- **Componentes**: Ícones geométricos, mockup com glassmorphism, linhas finas
